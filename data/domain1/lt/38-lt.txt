Civilinis procesas – tvarka, pagal kurią vyksta civilinių ginčų sprendimas teismo keliu. Bylas nagrinėja bendrosios kompetencijos teismas.


== Stadijos ==
Pagrindinė civilinio proceso dalis – nagrinėjimas teisme. Jame betarpiškai tiriami įrodymai. Po jų vertinimo priimamas teismo sprendimas. Juo ginčas yra išsprendžiamas iš esmės.
Paskutinė proceso stadija – vykdymo procesas. Atlieka antstoliai.


== Proceso šalys ==

Atsakovas - civilinio proceso dalyvis (asmuo, turintis teisinį suinteresuotumą bylos baigtimi), besiginančioji proceso šalis. Jam yra reiškiami reikalavimai (žr. Ieškinys), į kuriuos atsakovas gali atsakyti specialiais procesiniais dokumentais (žr. atsiliepimas, triplikas).
Priešinga proceso šalis - ieškovas.

== Civilinio proceso teisė ==
Civilinį procesą reguliuoja specialios proceso teisės normos – civilinio proceso teisė. Lietuvoje – Lietuvos Respublikos civilinio proceso kodeksas.
